**适用场景**: AI 应用开发者学习如何用 Claude Managed Agents 构建生产级多智能体系统，或通过评估驱动方法优化 Agent 行为。

**成熟度**: 非维护的培训材料，但内容官方且与最新 API 同步，适合学习而非生产。
