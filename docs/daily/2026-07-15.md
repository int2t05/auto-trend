- 代码解析零 LLM 成本，仅对文档/媒体使用可选 API，构建图几乎零费用

**核心功能**

- 使用 tree-sitter AST 进行代码解析，完全本地化、无需 LLM，保证数据隐私
- 每条边带有 EXTRACTED/INFERRED 置信标签，区分直接引用与推理连接
- 基于 Leiden 算法进行社区检测，自动拆分模块并生成可理解的子系统标签

**适用场景**: 使用 AI 编程助手（如 Claude Code、Cursor）的开发者，需要快速理解复杂代码库架构、追踪跨文件依赖时，可替代 grep 或手动阅读。

**竞品对比**: 对比 mem0 和 supermemory 等记忆/知识系统，Graphify 在 LOCOMO recall@10 上大幅领先（0.497 vs 0.048），且知识图构建无需消耗 LLM token。

**成熟度**: 生产可用，有 PyPI 包、多语言 README 和活跃社区，版本已迭代至 v8（参见文档图标），但 API 可能仍会变化。

**趋势信号**: 随着 AI 编程助手普及，开发者对代码库理解工具需求激增，Graphify 提供免 LLM 的本地知识图方案，切中隐私与成本痛点。

> 基于Tauri和Rust的开源视频编辑器，免费替代CapCut，支持硬件加速和AI功能。

**亮点**

- 采用Rust+FFmpeg后端而非WebCodecs，实现原生性能和浏览器无关性
- 解码器预热和顺序解码优化，将首帧延迟降至5-10ms

**核心功能**

- 基于FFmpeg硬件加速解码（VideoToolbox/D3D11VA/VAAPI）实现GPU视频处理
- 解码器预热和LRU解码池，首帧延迟低于10ms
- Web Worker池并行生成缩略图，减少主线程阻塞

**适用场景**: 需要专业视频编辑功能但不愿付费的个人创作者、教育机构、开源项目，替代CapCut等商业软件。

**竞品对比**: 相比OpenShot和Shotcut，Clypra拥有更现代UI和硬件加速优化；相比付费CapCut，完全免费开源。

**成熟度**: 早期项目（版本可能<1.0），但已有完整功能文档和CI，社区活跃度较高。

**趋势信号**: CapCut收费后用户寻找替代方案，加上Tauri v2和React 19的吸引力，引发大量关注。

### [par274/sharpemu](https://github.com/par274/sharpemu)

 `C#` ⭐ +2,190 今日新增 · 2,190 总星数

> 实验性PS5模拟器，C#从零编写，已能加载游戏并部分执行CPU指令。

**亮点**

- 以C#编写完整的系统模拟器，依赖.NET高级特性如Span<T>和内存映射，而非传统C++低级API
- 直接解析PlayGo场景格式，绕过文件系统直接处理加密资源

**核心功能**

- 全C#实现，利用.NET SDK构建，非传统C++模拟器路线
- 支持Fiber和AMPR导出，实现轻量级用户态线程调度
- PlayGo场景解析，处理游戏资源分块加载

**适用场景**: 适合PS5逆向工程研究者、模拟器开发爱好者验证CPU和图形管线；也供游戏开发者测试原生二进制在模拟环境中的行为。

**竞品对比**: 与Kyty（C++ PS5模拟器）相比，SharpEmu采用C#和跨平台.NET，但成熟度更低，目前仅少数游戏可进入视频输出阶段。

**成熟度**: 早期实验阶段，仅能运行部分游戏的初始加载，API与功能极不稳定。

**趋势信号**: PS5模拟器稀缺，项目首次公开后24小时内获得2190星，引发社区关注。
